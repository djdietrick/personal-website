Noise Monitor

Arduino node network equipped with microphones and radio transceivers that record sound level readings and report to a central RaspberryPi node. Reports are uploaded to a RESTful API built with Django and can be viewed on a webapp.

Technogies used: Python (Django), C (Arduino), SQLite, REST
